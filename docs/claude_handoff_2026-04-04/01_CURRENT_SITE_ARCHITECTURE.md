# Current Site Architecture

最後更新：2026-04-04

## 一句話定義

這是一個以「法庭實況還原、專業共構筆記、審核後公開文章」為核心的網站，不是一般新聞站，也不是傳統自由論壇。

## 目前網站的四層結構

### 1. 首頁層

- 路由：`/`
- 任務：
  - 說明平台定位
  - 承接主視覺與導流
  - 顯示還原筆記主入口
  - 逐步收攏其他子頁內容

### 2. 還原筆記／工作檯層

- 路由：
  - `/sessions`
  - `/sessions/[id]`
  - `/sessions/compose`
- 任務：
  - 場次總覽
  - 單場次還原筆記閱讀與工作檯入口
  - 跨場次工作檯入口

### 3. 公開文章層

- 路由：
  - `/forum`
  - `/forum/[id]`
- 任務：
  - 匯集經審核後公開的觀庭共構文章
  - 提供公開閱讀、留言、按讚

### 4. 支援知識層

- 路由：
  - `/about`
  - `/about-preview`
  - `/guide`
  - `/knowledge`
  - `/rankings`
  - `/contact`
- 任務：
  - 計畫敘事
  - 平台說明與規範
  - 先備知識
  - 排行榜
  - 聯絡／回報

## 目前最重要的頁面關係

### `/sessions`

這頁已不是單純列表頁，而是：

- 還原筆記總覽
- 已發布完整筆記入口
- 跨場次工作檯入口
- 準備程序庭與審理程序庭整理

### `/sessions/[id]`

這是核心單頁，承接：

- 還原筆記正文
- 後續工作檯／論述形成
- 單篇引用與閱讀

### `/sessions/compose`

這是跨場次工作檯，目前屬骨架已在、後續仍要持續打磨的頁面。

### `/forum`

這不是傳統論壇，而是：

- 已審核共構文章的公開層
- 保留留言與互動
- 公開閱讀為主

## 共用外殼

### 共用子頁頁首

元件：

- `components/SubpageHeader.tsx`

目前定位：

- 所有主要子頁的共通頁首工版
- 不再有「我要投稿」按鈕
- 導覽列以正式頁面為主

### `/sessions` 內容共用區塊

元件：

- `components/SessionsOverviewSection.tsx`

目前定位：

- 承接 `/sessions` 的主要內容
- 也已開始被抽進首頁做內容收攏
- 是之後還原筆記總覽區的共用內容骨架
