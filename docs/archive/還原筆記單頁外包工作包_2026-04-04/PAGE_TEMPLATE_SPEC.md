# 還原筆記單頁規格

## 頁面定位

這個單頁是「單一場次的還原筆記工作檯」。

目標：

- 左側呈現逐字稿 / 類逐字內容
- 右側提供共構筆記編輯區
- 支援引用
- 支援從左側摘述內容到右側形成論述

## 頁面組成

### 1. 頁首

需包含：

- 返回場次列表
- 場次標題
- 場次代碼
- 分享按鈕
- 送出審核按鈕

### 2. 左側：逐字內容區

需包含：

- 場次基本資料
- 各段逐字稿 / 類逐字稿
- 每段可點擊
- 每段有固定 line id

### 3. 右側：共構文章編輯區

需包含：

- 文章標題欄
- 作者欄
- 聯絡信箱欄
- 編輯器
- 可插入引用標記

### 4. 引用機制

引用資料最少需保留：

- `lineId`
- `sessionId`
- `speaker`
- `label`

## 外包 AI 可協助的部分

- 製作場次 metadata
- 整理 transcripts 資料
- 確保每段有穩定 id
- 將原始逐字稿整理成頁面可用資料

## 外包 AI 不需處理的部分

- Tiptap / editor 邏輯
- React 元件互動
- API 投稿流程
- Notion 審查流程
